# 开发阶段操作指南 2.0

这份文档只保留每天最常用的命令，直接复制到终端即可。

---

## 终端 1：启动 MinIO（本地启动，不用 Docker）

```powershell
cd E:\minio
.\minio.exe server E:\minio-data --console-address ":9001"
```

---

## 终端 2：启动后端

```powershell
cd E:\blog\weblog\weblog-springboot
mvn install -DskipTests -pl weblog-web -am
java -jar E:\blog\weblog\weblog-springboot\weblog-web\target\weblog-web-0.0.1-SNAPSHOT.jar --spring.profiles.active=dev
```

如果提示 8080 被占用，就先输入：

```powershell
netstat -ano | findstr :8080
```

再结束占用 8080 的进程：

```powershell
taskkill /PID 进程号 /F
```

---

## 终端 3：启动前端

```powershell
cd E:\blog\weblog\weblog-vue3
npm run dev
```

---

## 提交并推送到远程仓库

直接复制下面这 4 行：

```powershell
cd E:\blog\weblog
git add .
git commit -m "本次修改说明"
git push
```

你平时只需要改这一句里的提交说明：

```powershell
git commit -m "本次修改说明"
```

比如可以改成：

```powershell
git commit -m "完善优化需求分析文档"
```

如果这次没有新改动，`git commit` 会提示没有可提交内容，这是正常的。
