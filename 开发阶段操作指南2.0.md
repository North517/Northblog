# 开发阶段操作指南 2.0

这份文档只保留每天最常用的 3 个终端启动命令。

---

## 终端 1：启动 MinIO（本地启动，不用 Docker）

先确保你本地已经有 `minio.exe`。

比如你可以放在：

```text
E:\minio\minio.exe
```

然后终端输入：

```powershell
cd E:\minio
.\minio.exe server E:\minio-data --console-address ":9001"
```

---

## 终端 2：启动后端

```powershell
cd E:\blog\weblog\weblog-springboot
mvn install -DskipTests -pl weblog-web -am
java -jar E:\blog\weblog\weblog-springboot\weblog-web\target\weblog-web-0.0.1-SNAPSHOT.jar --spring.profiles.active=dev
```

如果提示 8080 被占用，就先输入：

```powershell
netstat -ano | findstr :8080
```

再结束占用 8080 的进程：

```powershell
taskkill /PID 进程号 /F
```

---

## 终端 3：启动前端

```powershell
cd E:\blog\weblog\weblog-vue3
npm run dev
```
